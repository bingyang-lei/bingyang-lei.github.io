Draft-OPD: On-Policy Distillation for Speculative Draft Models

Draft-OPD is a post-training framework for speculative draft models. This temporary project page will host the paper, code, models, figures, and additional resources.

GitHub Code HuggingFace Models

Overview

Speculative decoding accelerates large language model inference by verifying draft-model proposals in parallel. Draft-OPD improves draft models by training them on verification-time states and errors exposed during online drafting.
